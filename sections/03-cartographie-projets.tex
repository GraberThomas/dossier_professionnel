\section{Logique de sélection des projets}

Le dossier s'appuie sur une sélection de projets professionnels, académiques et complémentaires. Le projet FCU / Ediacara constitue le projet principal, car il permet de démontrer une démarche complète : analyse d'un existant complexe, adaptation technique, tests, mise en production, documentation et transmission de connaissances.

Les autres projets sont mobilisés pour compléter la couverture du référentiel, notamment sur le développement back-end, le développement front-end, le traitement de données, le DevOps et le pilotage d'équipe.
