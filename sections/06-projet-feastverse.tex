\section{Contexte et objectifs du projet}

Ce chapitre présentera le projet FeastVerse, retenu pour démontrer une maîtrise du développement back-end applicatif à travers une API REST Java Spring Boot.

\section{Compétences principalement mobilisées}

Les compétences principalement visées sont : C1.3, C2.1, C2.2, C2.4, C3.2 et C3.6.

\section{Plan de rédaction prévu}

Ce chapitre développera l'architecture Spring Boot, le modèle de données, l'ORM, l'authentification, les rôles et permissions, les endpoints REST, les tests Postman, la documentation et le bilan critique.
